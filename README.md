# 🛠 Corevia Global — Role Manager System
A modular, cinematic, permission‑checked role management system built for Discord servers that demand structure, hierarchy, and automation.

Corevia Global’s Role Manager provides a clean, expandable command suite for assigning roles, managing role managers, and enforcing authority with precision.

## Features
- Add / Remove roles with permission‑checked validation  
- Temporary roles with automatic expiration  
- View role metadata, managers, and hierarchy  
- Role Manager System (assign & remove managers)  
- Modular command structure  
- Secure, permission‑based execution  
- Cinematic Corevia‑style embeds
